# 智能助手团子 - 前端页面说明

## 项目概述

这是一个现代化的聊天界面，用于与基于 Qwen3 模型的 AI 助手进行交互。界面采用现代化设计，具有流畅的用户体验。

## 设计特色

- **现代化UI设计**：渐变色彩搭配，圆润的卡片式设计
- **响应式布局**：适配不同屏幕尺寸
- **动效体验**：消息气泡悬停效果、平滑过渡动画
- **本地历史记录**：自动保存聊天记录到浏览器本地存储
- **清空功能**：一键清空当前聊天记录
- **帮助文档**：内置使用指南，便于用户快速上手

## 技术特性

- **CSS3 渐变背景**：使用线性渐变和径向渐变创造视觉层次
- **CSS3 动画效果**：消息淡入动画、悬停变换效果
- **ES6 JavaScript**：现代化语法，模块化设计
- **Local Storage**：浏览器本地数据持久化
- **Fetch API**：现代化异步请求处理

## 使用说明

1. 启动后端 Spring Boot 应用程序
2. 确保 Ollama 服务正在运行，并已加载 qwen3:0.6b 模型
3. 访问 `http://localhost:8080` 进入聊天界面
4. 在输入框中输入问题，点击发送或按回车键
5. 点击"清空"按钮可清除当前对话记录
6. 点击右上角"帮助"查看详细使用指南

## 设计灵感

界面设计参考了豆包（Doubao）等现代 AI 助手的 UI 特色，采用了类似的对话气泡设计、渐变色彩方案和交互模式，同时加入了自己的创新元素。
