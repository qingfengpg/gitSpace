智能助手团子 - 使用帮助

欢迎使用基于Qwen3模型的智能聊天助手

欢迎使用

智能助手团子是一个基于先进AI技术的聊天机器人，使用阿里巴巴通义千问Qwen3模型为您提供智能对话服务。

团子可以回答问题、创作文字、表达观点、玩游戏等，是您的智能助手。

主要功能

  • 智能问答 - 提出任何问题，团子都会尽力回答
  • 创意写作 - 让团子帮您写故事、公文、邮件等
  • 编程辅助 - 提供编程建议和代码示例
  • 语言翻译 - 支持多种语言互译
  • 聊天陪伴 - 日常闲聊，情感支持
  • 知识学习 - 解释概念，提供学习资料

使用方法

开始聊天：在输入框中输入您的问题或话题，点击"发送"按钮或按回车键即可开始对话。

清空记录：点击"清空"按钮可清除当前聊天记录，但不会影响历史记录。

快捷键：按回车键快速发送消息（Shift+回车可换行）。

使用技巧

清晰表达：尽量使用清晰明确的语言提问，这样团子能更好地理解您的需求。

上下文关联：在连续对话中，团子会记住之前的对话内容，您可以自然地延续话题。

具体问题：对于复杂问题，尽量提供更多背景信息，以获得更准确的回答。
